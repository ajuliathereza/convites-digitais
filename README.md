# Convite Digital Animado — Joaquim 🦁🌿

Projeto de **convite digital interativo** com temática **savana / rei leão**, com:
- topo com **vídeo** (com fallback),
- **envelope animado** (abre a tampa + carta sobe),
- dados do evento (dia, horário, local),
- botões de ação (WhatsApp, Maps, presentes),
- modal de presentes,
- contagem regressiva,
- design responsivo e acessível (focus/`prefers-reduced-motion`).

> Ideal para: aniversários infantis, chás, batizados e eventos com tema.

---

## 📁 Estrutura do projeto

```
.
├─ index.html        # Estrutura do convite
├─ style.css         # Estilos (layout, animações, responsivo)
├─ script.js         # Configuração do convite + interações
├─ videos/           # Vídeo do topo (ex.: reileao.mp4)
└─ images/           # Imagens (ex.: selo/adesivo do envelope)

```
